Setup Notes

Google Form fields to create

Use these exact question names in your Google Form so the page can read them cleanly:

  • Round
  • Court
  • Team A
  • Team B
  • Points A
  • Points B

Optional fields are fine, but keep these six exactly as above.

How to publish the Sheet as CSV

  1. Open the Google Sheet that receives the form responses.
  2. Go to File → Share → Publish to web.
  3. Choose the responses tab.
  4. Select CSV.
  5. Paste that URL into GOOGLE_SHEET_CSV_URL in the config below.

Important tournament logic in this page

  • 8 teams
  • 3 courts
  • 4 Swiss rounds
  • 2 byes per round
  • Points are the actual games scored, not win/loss points
  • Standings sort by Total Points, then Game Difference, then Strength rating
  • The pairing engine tries to avoid repeat matches and spread byes fairly
